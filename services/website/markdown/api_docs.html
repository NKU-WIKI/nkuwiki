南开Wiki API文档

更新记录

  • 2023-03-26: 增加RAG（检索增强生成）相关接口（8.4-8.5），添加suggested_questions字段到返回结果
  • 2024-03-27: 更新API路径，添加"api/"前缀；移除已废除的MySQL接口

本文档包含南开Wiki平台的所有API接口，主要分为两类： 1. 微信小程序API：提供给微信小程序客户端使用的API 2. Agent智能体API：提供AI聊天和知识检索功能

接口前缀

所有API都有对应的前缀路径： - 微信小程序API：/api/wxapp/* - Agent智能体API：/api/agent/*

如，用户接口的完整路径为 /api/wxapp/users/me

后端响应标准格式：

{
  "code": 200,
  "message": "success",
  "data": {
    "key1": value1
  }
}

参数名 类型 必填 默认值 说明
query string 是 - 用户查询问题
tables array 否 ["wxapp_posts"] 要检索的表名列表
max_results int 否 5 每个表返回的最大结果数
stream bool 否 false 是否流式返回
format string 否 "markdown" 回复格式: markdown/text/html
openid string 否 null 用户唯一标识
