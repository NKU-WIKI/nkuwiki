南开Wiki RAG接口文档

文档导航

南开Wiki检索增强生成(RAG)接口文档集合，用于开发者了解和使用RAG服务。

核心文档

  • RAG API接口文档 - 详细的API接口说明，包含参数、响应格式和示例
  • RAG接口使用指南 - 开发者使用指南，包含最佳实践和示例代码
  • RAG接口测试报告 - 测试结果和性能报告

快速开始

对于希望快速集成RAG接口的开发者，建议先阅读RAG接口使用指南，了解基本概念和使用方式。

需要详细API参数和响应格式的开发者，可以参考RAG API接口文档。

测试环境

我们提供了一个简化的测试环境：

http://localhost:8888/rag

该测试环境实现了与正式环境相同的API接口，但使用了模拟数据，非常适合前端开发和接口集成测试。
